# C Programming

This repository contains a structured collection of C programs designed to teach C programming from basic to advanced levels.  
It is intended for students, beginners, and educators using **MinGW GCC** and **Code::Blocks IDE** on Windows.

## Objectives

- Build strong fundamentals in C programming
- Understand core concepts through simple and clear examples
- Improve logical thinking and problem-solving skills
- Prepare learners for system programming, embedded systems, and further study in computer science

## Tools and Environment

- Programming Language: C (ANSI C / C99)
- Compiler: MinGW (GCC)
- IDE: Code::Blocks
- Platform: Windows

## Topics Covered

### Beginner Level
- Introduction to C
- Variables and Data Types
- Input and Output
- Operators
- Control Statements (if, switch, loops)

### Intermediate Level
- Functions and Recursion
- Arrays and Strings
- Pointers
- Structures and Unions
- File Handling

### Advanced Level
- Dynamic Memory Allocation
- Preprocessor Directives and Macros
- Typedef and Enum
- Memory Management Concepts

## How to Run the Programs

### Using Code::Blocks
1. Open Code::Blocks
2. Create a new empty project
3. Add the required `.c` file to the project
4. Build and run the project

### Practice Platform C
Codechef : https://www.codechef.com/learn/course/c

### Teaching Approach

Each program focuses on a single concept
Code is written in a simple and readable manner
Programs are suitable for classroom teaching and self-learning
Emphasis is placed on understanding fundamentals before advanced topics
